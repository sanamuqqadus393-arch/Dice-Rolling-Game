# Dice Rolling Game (Python)

## Overview

This project is a simple Dice Rolling Game built using Python.
The program simulates rolling one or more dice and displays random values between 1 and 6 for each dice.

The user can choose how many dice to roll and can continue playing multiple times until they decide to exit the game.

## Features

- Roll multiple dice at once
- Random dice values between 1 and 6
- Tracks how many times the dice has been rolled
- User option to continue or stop the game
- Simple command line interface

## Technologies Used

- Python
- Random module

## How the Game Works

1. The program starts with a welcome message.
2. The user is asked how many dice they want to roll.
3. The program generates a random number (1 to 6) for each dice.
4. The result of each dice is displayed.
5. The program counts how many times the user rolled the dice.
6. The user can choose to roll again or exit the game.

## Example Output

How many dice do you want to roll? 2

Dice 1 : 3
Dice 2 : 6

Roll the dice again? (y/n): y

## How to Run the Project

1. Install Python.
2. Download or clone this repository.
3. Open the project folder in VS Code or any Python editor.
4. Run the Python file.

## Example command:

python Dice_Rolling_Game.py
